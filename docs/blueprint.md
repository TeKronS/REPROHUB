# **App Name**: Muralis

## Core Features:

- Cargador de Imágenes: Interfaz de arrastrar y soltar con validación de formato para activos de impresión de alta calidad.
- Distribución en Cuadrícula: Lienzo interactivo para dividir una imagen en múltiples filas y columnas según tamaños de papel estándar.
- Controles de Escala y Margen: Ajustes para el solapamiento, márgenes de pegado y dimensiones físicas de impresión.
- Motor de Exportación PDF: Generación de un PDF paginado con guías numeradas para facilitar el montaje físico.
- Vista Previa de Maqueta: Simulación visual de los paneles impresos finales en una pared a escala real.
- Soporte Multi-idioma: Selector para alternar la interfaz entre Español e Inglés de forma instantánea.

## Style Guidelines:

- Color primario: Verde Esmeralda Vibrante (#2E7D32) para una estética orgánica y profesional.
- Color de fondo: Bosque Profundo (#0A140B) para un modo oscuro elegante y de bajo cansancio visual.
- Color de acento: Verde Lima Neón (#C6FF00) para resaltar botones de acción y estados activos.
- Fuente de encabezado: 'Space Grotesk' para un estilo técnico; Fuente de cuerpo: 'Inter' para máxima legibilidad.
- Iconos minimalistas de trazo fino (2px) que representan medidas, cuadrículas y hojas.
- Diseño de suite creativa con lienzo central y paneles de ajuste modulares a la derecha.
- Microinteracciones fluidas para la transición de paneles y efectos de manipulación de cuadrícula.
